Overview
================================================================================

On this guide you will learn how to deploy Clox by the hard way. It is not meant for normal users. If you only want to set Clox up, please move to :ref:`The short road <the_short_road_overview>`. If you are an advanced user, want to know how everything was made and even contribute, this is the right place for you.
